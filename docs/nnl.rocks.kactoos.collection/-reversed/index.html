Reversed

class Reversed<X : Any> : CollectionEnvelope<X> (source)

Reversed collection.

Pay attention that sorting will happen on each operation with the collection. Every time you touch it, it will fetch the entire collection from the encapsulated object and reverse it. If you want to avoid that "side-effect", decorate it with StickyCollection.

There is no thread-safety guarantee.
