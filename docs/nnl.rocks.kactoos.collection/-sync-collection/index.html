SyncCollection

class SyncCollection<T : Any> : CollectionEnvelope<T> (source)

Iterable as Collection.

This class should be used very carefully. You must understand that it will fetch the entire content of the encapsulated Iterable on each method call. It doesn't cache the data anyhow. If you don't need this Collection to re-fresh its content on every call, by doing round-trips to the encapsulated iterable, use StickyCollection.

Objects of this class are thread-safe.
