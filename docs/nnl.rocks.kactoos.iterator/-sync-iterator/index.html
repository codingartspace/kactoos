SyncIterator

class SyncIterator<out T> : Iterator<T> (source)

Synchronized Iterator implementation using a ReadWriteLock either provided to the constructor or an internally created ReentrantReadWriteLock. The ReadWriteLock is used to synchronize read calls to SyncIterator.hasNext against write calls to SyncIterator.next and write calls to any other read or write calls.

Parameters

T - The type of the iterator.

iterator - The iterator to synchronize access to.

lock - The lock to use for synchronization.
