SyncList

class SyncList<X : Any> : ListEnvelope<X> (source)

Synchronized list.

This class should be used very carefully. You must understand that it will fetch the entire content of the encapsulated Iterable on each method call. It doesn't cache the data anyhow. If you don't need this java.util.List to re-fresh its content on every call, by doing round-trips to the encapsulated iterable, use StickyList.

The list is read only.

Objects of this class are thread-safe.
