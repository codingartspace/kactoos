# &lt;init&gt;

`Equality(left: `[`T`](index.md#T)`, right: `[`T`](index.md#T)`)`

Equality.

Returns:

* the value `0` if `x == y`;
* the value `-1` if `x < y`;
* the value `1` if `x > y`

There is no thread-safety guarantee.

### Parameters

`T` - Type of input

`left` - Left

`right` - Right

0.4
