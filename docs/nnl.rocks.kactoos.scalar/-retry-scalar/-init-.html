Func that will try a few times before throwing an exception.

There is no thread-safety guarantee.

This class implements Scalar, which throws a checked Exception. This may not be convenient in many cases. To make it more convenient and get rid of the checked exception you can use UncheckedScalar or IoCheckedScalar decorators.

Parameters

T - Type of output

origin - Func original

func - Exit condition, returns TRUE if there is no reason to try
